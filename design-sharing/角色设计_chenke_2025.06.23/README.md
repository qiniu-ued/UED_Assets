# 角色设计

## 人物设定
- **姓名**：艾洛斯·敖特沃  
- **性别**：女  
- **年龄**：23  
- **职业**：狙击手  
- **身份**：国家联合防疫部门特级人员  

## 外貌设定
- 粉色头发（家族象征）
- 青色眼眸
- 白色衬衣 + 粉色领带
- 肩部徽章：特级人员身份认证，同时作为部门通行证
- 腿环：装有预备子弹（用于防止尸变的特殊子弹）

## 世界背景
- 时间：4068 年  
- 新型传染病毒再次爆发  
- 病毒特性：
  - 从破坏人体生理结构进化为可控制人类大脑  
  - 利用人体机能为自身提供营养  
- 病毒传播速度极快  
- 国家联合部门紧急组织防疫与围剿行动  
- 所有配置子弹均由特殊研究血清制作，用于对抗病毒与防止尸变  

## 角色背景
- 艾洛斯性格内向，因此培养出理智、严谨的处事方式  
- 不擅长与人交流，但沉着冷静  
- 冷静性格是其成为特级狙击手的重要因素之一  
- 家境优渥，但因是次女，父母关注较少  
- 在敖特沃家族中并不突出  
- 16 岁加入国家特训部门，参与集中训练  
- 凭借自身刻苦努力进入国家联合防疫部门  
- 成为史上第一位女性狙击手  
- 敖特沃家族的超群视力得到传承，是其成功的重要因素之一  

## 角色概念想法
- 初始概念意象：蝴蝶  
- 演变方向：
  - 翅膀变成枪支  
  - 蝴蝶随风翩翩起舞，演变为角色攻击时的动态表现
